    ModuleParser

    new ModuleParser()

    The ModuleParser is a utility class designed to loop through and iterate
    on a directory and pull out of each .js file found, any classes or exports
    that extend from GQLBase or a child of GQLBase.

    Since:
    • 2.7.0
